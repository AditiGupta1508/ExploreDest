# ExploreDest — Student Budget Trip Planner

A full-featured trip planning web app built for students who want to travel smart.

## 🔗 Live Demo
[View Live Site](https://AditiGupta1508.github.io/ExploreDest)

##  Features
-  GPS auto-detection via browser geolocation
-  Interactive Leaflet.js map with live route drawing
-  Smart budget algorithm using Haversine distance formula
-  7-day live weather forecasting (Open-Meteo API)
-  Real-time currency conversion (INR/USD/EUR)
-  Wikipedia API for automatic destination images
-  Hidden gems, itinerary builder, and financial breakdown

##  Tech Stack
HTML · CSS · Vanilla JavaScript · Leaflet.js · REST APIs

##  APIs Used
- Nominatim (OpenStreetMap) — Geocoding
- Open-Meteo — Weather forecasting  
- Exchange Rate API — Currency conversion
- Wikipedia API — Destination images
- Browser Geolocation API — GPS detection
